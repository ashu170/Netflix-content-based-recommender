# 🎬 Netflix Movie Recommender System

A Content-Based Movie Recommendation System that recommends similar Netflix movies using Machine Learning.

## 🚀 Features
- Content-Based Recommendation
- Cosine Similarity
- Movie Poster Fetching using TMDB API
- Streamlit Web Application
- Fast and Interactive UI

## 🛠 Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- TMDB API
- Pickle

## 📂 Project Structure

```
Netflix-Movie-Recommender/
│── app.py
│── movie_recommender_system.ipynb
│── movies.pkl
│── similarity.pkl
│── requirements.txt
│── README.md
```

## ⚙️ Installation

```bash
git clone https://github.com/ashu170/netflix-content-based-recommender.git

cd netflix-content-based-recommender

pip install -r requirements.txt

streamlit run app.py
```
